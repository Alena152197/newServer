# Фронтенд приложения

## Структура файлов

- `index.html` - страница входа
- `register.html` - страница регистрации  
- `dashboard.html` - страница профиля (после входа)
- `js/auth.js` - JavaScript для работы с API
- `css/input.css` - исходный файл Tailwind CSS
- `css/output.css` - скомпилированный CSS (production)

## Использование

1. Запустите сервер: `go run main.go` или `./server.exe`
2. Откройте браузер: `http://localhost:4000` (порт по умолчанию)
3. Вы увидите страницу входа

## Примечания

- По умолчанию сервер работает на порту **4000** (не 8080)
- Если нужно изменить порт, установите переменную окружения `PORT`
- Все статические файлы обслуживаются из папки `client/`
- Для перекомпиляции CSS: `npm run build-css`
